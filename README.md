## ✨ Requirements
* Operating System (MacOS X, Linux, Windows)
* IDE with Flutter SDK installed (Android Studio, VSCode, IntelliJ)

## 📱 App
* Made with Dart and Flutter
* Api Integration (PHP and MySQL)
* Login Register
* Localization
* OTP
* Payment
* CRUD Operation
